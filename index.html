Bunny's Trouble

Bunny's Trouble was created by myself within a week for Brackey's second Game Jam in 2021 using the Godot engine. In a shoot-em-up style, the player has to defeat waves of enemies that get increasingly more bullet-hell-y. After each wave you are rewarded with a power-up, but be careful, for each hit you also lose a power-up!

  • Everything was created by me for the event!

Desktop Disaster

Desktop Disaster is a little game developed in Unity for the Ludum Dare 42 Game Jam, where an anthropomorphic virus takes over your computer and keeps creating files, trying to get it to carsh. The player needs to analyze each file that gets dragged onto the desktop and decide if it's a good file or a bad file. The virus however, does not wait so you better be quick, before your storage runs out!

  • Code the game
  • Music / Sounds
  • Implementation in Unity
